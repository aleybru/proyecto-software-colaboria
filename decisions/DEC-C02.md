# DEC-C02 — Dos repositorios separados: gobernanza y código

**Estado:** Decision vigente

## Resumen
El proyecto usa dos repositorios de GitHub separados:
- `aleybru/proyecto-software-colaboria` — repositorio de gobernanza (decisiones, arquitectura, handoffs, workspace).
- `aleybru/proyecto-software-colaboria-codigo` — repositorio de código (skeleton de la app).
